cy_stc_autanalog_cfg_t Struct Reference
AUTONOMOUS ANALOG (Autonomous Analog Block) » General section of the API » Data Structures

Description

The configuration structure to set up the entire Autonomous Analog used with cy_stc_autanalog_t and Cy_AutAnalog_LoadConfig.

For more details, refer to the device Architecture Technical Reference Manual.
