AUTONOMOUS ANALOG (Autonomous Analog Block) » CT DAC (Continuous Time Digital to Analog Converter) » Functions

General Description

This set of functions is used to configure the DAC.

Note
The configuration must be done only under the AC in High Speed mode, when cy_stc_autanalog_stt_ac_t::lpMode is equal to FALSE.
