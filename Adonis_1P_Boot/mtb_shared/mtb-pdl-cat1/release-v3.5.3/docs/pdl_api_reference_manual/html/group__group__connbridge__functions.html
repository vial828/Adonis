uint32_t  intrNum,
uint32_t  value 
)

Configures the interrupt line to be forwarded to the CPU NVIC.

Parameters
basePointer to the mxconnbridge register base address
intrNumIncoming interrupt line number.
value0 = interrupt not forwarded to CPU interrupt controller 1 = interrupt masked and forwarded to CPU interrupt controller
Note
This function modifies a register in a read-modify-write operation. It is not thread safe as the resource is shared among multiple interrupt lines.
Function Usage
