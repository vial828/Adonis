Description

I3C device structure.

Defines an I3C device. StaticAddress of the I3C device if provided will be used by the controller to initiate the dynamic address assignment procedure through SETDASA. Desired I3C dynamic address to be used can be provided, but if set to zero, available free address will be assigned to the device by the controller.
