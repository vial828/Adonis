AUTONOMOUS ANALOG (Autonomous Analog Block) » CTB (Continuous Time Block) » Data Structures

Description

The CTB section in the State Transition Table state.

Used in cy_stc_autanalog_stt_t. For more details, refer to the device Architecture Technical Reference Manual.

Data Fields

bool unlock
 CTB Unlock
FALSE - Locked:
Data from fields enableOpampX, cfgOpampX, gainOpampX will NOT be taken into account in the corresponding state of STT
TRUE - Unlocked:
Data from fields enableOpampX, cfgOpampX, gainOpampX will be used in the STT corresponding state.
