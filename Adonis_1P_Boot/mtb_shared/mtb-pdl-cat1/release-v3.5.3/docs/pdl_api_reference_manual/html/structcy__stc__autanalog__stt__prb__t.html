AUTONOMOUS ANALOG (Autonomous Analog Block) » General section of the API » Data Structures

Description

The PRB section in the State Transition Table state, used in cy_stc_autanalog_stt_t.

For more details, refer to the device Architecture Technical Reference Manual.

Data Fields

bool unlock
 PRB Unlock:
FALSE - Locked:
Data from fields prbVrefxFw and prbVrefxTap will NOT be taken into account in the corresponding state of STT
TRUE - Unlocked:
Data from fields prbVrefxFw and prbVrefxTap will be used in the STT corresponding state.
