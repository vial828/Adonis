cy_nn_dims_t Struct Reference
NNLITE HW Accelerator KERNEL LIB » Structures

Description

dimension structure for input, output & filter dimension array of tensor, ordering of dimension should be same as TF model file, e.g.

for convolution layer input height and width order should be input height = inputDims->dims[1], input width = inputDims->dims[2], input channel = inputDims->dims[3], filter dimension order should be filter_height = filterDims->dims[1], filter_width = filterDims->dims[2],filter_depth = filterDims->dims[3], output dimension order should be output_height = outputDims->dims[1], output_width = outputDims->dims[2] activationRepeats = MIN(outputDims->dims[3], filterDims->dims[0]) for fully connected layer input height and channel are always =1 inputWidth = inputDims->dims[1] filter height is always 1, filter width = filterDims->dims[1] activationRepeats = MIN(filterDims->dims[0], outputDims->dims[1]), rest of the dimensions are not used Average pool is implemented using convlution software dimensions order should be same as convolution
